# 💫 About Me:

Hey, I'm **Ren**.

I'm mostly a **Python enjoyer** who likes building things just to see how they work and how they can be improved. I spend a lot of time experimenting with ideas, refactoring code that annoyed me yesterday, and occasionally creating something that turns out to be genuinely useful.

Most of what lives on this profile is basically **structured tinkering**; small projects, experiments, and attempts to make code cleaner and logic less questionable.
Frontend is... not really my thing, so I usually hide in the **backend and core logic** where things make more sense.

Outside of coding I spend time with:

* **Chess and other strategy games**
* **Anime and manga** :3
* **Problem solving**
* **Math, philosophy, and whatever topic steals my curiosity that week**

This profile isn't really a showcase, it's more of a **trail of experiments and improvements**.
If something is here, it probably taught me something. And if it didn't… well, at least it looked interesting at the time.
